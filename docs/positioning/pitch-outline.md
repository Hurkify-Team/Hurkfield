# Pitch Outline

## 1. Problem
Field data collection is slow, fragile, and hard to supervise.

## 2. Solution
OpenField is a research operations platform for field data collection.

## 3. Differentiators
- Project-first hierarchy
- Assignment-scoped share links
- Ethics controls and QA

## 4. How it works
- Build template
- Assign enumerators
- Collect submissions
- Review QA
- Export for analysis

## 5. Traction plan
- Run a pilot
- Capture feedback
- Iterate on blockers

## 6. Ask
- Pilot partner and feedback
