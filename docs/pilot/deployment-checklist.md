# Deployment Checklist (First Live Pilot)

## Before pilot
- Confirm project scope and timeline
- Confirm template questions and required fields
- Confirm coverage structure
- Add enumerators and assignment links
- Share enumerator guidance

## During pilot
- Monitor submissions daily
- Check QA Alerts for missing data
- Log blockers in feedback log

## After pilot
- Export final dataset
- Conduct debrief
- List top 5 fixes
- Decide go/no-go for next pilot
