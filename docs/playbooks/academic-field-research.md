# Pilot Playbook: Academic Field Research

Goal: structured, ethics-aware data collection for research teams.

## Prebuilt project
- Name: Academic Field Research Pilot
- Status: Active
- Description: Ethics-first data collection with attestation.

## Template (Form) questions
Section: Ethics
Consent and enumerator attestation are collected via the built-in ethics prompts (required).

Section: Respondent profile
- Participant code (text, optional)
- Respondent age (number, required)
- Education level (dropdown: None, Primary, Secondary, Tertiary)
- Employment status (dropdown: Employed, Self-employed, Unemployed, Student)

Section: Research module
- Q1: Primary outcome (text, required)
- Q2: Secondary outcome (text)
- Q3: Follow-up needed? (yes/no)

Section: Notes
- Researcher notes (long text)

## Suggested coverage structure
Country -> Region -> Site -> Participant

## Enumerator assignment
- Enumerator name and optional code
- Optional coverage scope: site or participant
- Use assignment-scoped share links

## Anonymized responses
- Do not collect names; use participant codes instead
- Sensitive exports can be shared without identifiers

## Methodology documentation
- Reference the consent + attestation flow in your methods section
- Save exports and QA logs as appendix material

## Success criteria
- Consent rate above 90 percent
- Attestation completed for all submissions
- Clean export for analysis
