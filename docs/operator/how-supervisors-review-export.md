# How Supervisors Review and Export

This is for supervisors managing quality and exports.

## Review submissions
- Go to Submissions
- Open a submission to review details
- Use QA Alerts for missing data or inconsistencies

## Fix issues
- If there are errors, contact the enumerator to re-submit
- Track system errors in Errors if needed

## Export data
- Go to Exports
- Optionally filter by project, template, or date
- Download CSV for SPSS/Stata/Excel or JSON for analysis

## Keep a record
- Save exports with date in filename
- Store in project folder with notes
