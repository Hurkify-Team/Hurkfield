# How Enumerators Collect Data

This is a quick guide for field teams.

## Step 1: Open the link
- Supervisor sends a share link
- If assigned, use the assignment-specific link

## Step 2: Fill the form
- Required questions are marked
- Use Next to move between sections
- If a required item is missing, the form will highlight it
- If GPS is enabled, allow location when prompted

## Step 3: Review and submit
- Review the summary section
- Tap Submit
- Confirm success message

## If you lose connection
- Keep the page open and continue when connected
- If enabled, use Save Draft and resume later

## Common mistakes to avoid
- Skipping required fields
- Using the wrong coverage location
- Submitting without consent
